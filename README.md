# DerridaHiggs
Derrida Higgs model with finite genome

Files:

1) "DH-Bc-Heuristic.nb" (Wolfram Mathematica notebook) contains the heuristic solution proposed in "Marquioni, V. M., & de Aguiar, M. A. (2025). The transition to speciation in the finite genome Derrida–Higgs model: a heuristic solution. Journal of Physics A: Mathematical and Theoretical, 58(17), 175601."

  This file also generates Figure 6.4 from the thesis: "MONTEIRO, Vítor Marquioni. Applications of the finite Derrida-Higgs model to population dynamics: Aplicações do modelo de Derrida-Higgs finito em dinâmica de populações. 2024. 1 recurso online (210 p.) Tese (doutorado) - Universidade Estadual de Campinas, Instituto de Física Gleb Wataghin, Campinas, SP."
  
2) "Bc_mu.txt" : the heuristic solution data in Figure 4 (top panel)

3) "Bc_N.txt" : the heuristic solution data in Figure 4 (middle panel)

4) "Bc_mu.txt" : the heuristic solution data in Figure 4 (bottom panel)
